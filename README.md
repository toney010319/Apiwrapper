# Name Prediction API

### Determine Age
```
HTTP METHOD: GET
URL: http://localhost:3000/age?name=michael
```
Parameters
| Name    |     Description       | Required |
|---------|-----------------------|----------|
| `name`  | firstname of user     | true     |
